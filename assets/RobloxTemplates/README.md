# Roblox Template Assets

Place the canonical Roblox R15 base rig FBX file in this directory using the exact name `R15_Base.fbx`.

The Phase 1-5 pipeline expects:
- Mesh scale in meters (1 Roblox stud = 0.28 m)
- Bone hierarchy that matches Roblox Studio's default R15 mapping
- Neutral pose (arms down, palms in)

After downloading from Roblox documentation, drop it here before running the Blender automation scripts.
